# Getting Started

## Prerequisites
- Node.js 18+
- npm

## Installation

```bash
git clone https://github.com/mk-knight23/03-studio-site.git
cd 03-studio-site
npm install
npm run dev
```

## Customization

### Branding
- Update logo URL in `index.html`
- Change colors in `src/style.css`
- Replace videos with your own

### Content
- Edit text in `index.html`
- Update client names and images
- Add your own case studies

### Animations
- Modify GSAP timelines in `src/main.ts`
- Adjust scroll trigger points
- Customize cursor behavior

## Development

```bash
npm run dev      # Start dev server
npm run build    # Build for production
npm run preview  # Preview build
```
